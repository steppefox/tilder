# TILDER

Translations Manager

# Requirements

* Node.js >= 8 - We can suggest to use NVM [https://github.com/creationix/nvm](https://github.com/creationix/nvm) for that

# Installation Dev

* Clone repository
* `npm install`
* `npm run dev`
* Open `http://localhost:3000`
